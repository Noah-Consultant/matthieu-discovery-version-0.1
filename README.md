## Modules consulting proposés

```
Module A — Pilote "Excel & data-prep"       →  P1 + P2  (quick win)
Module B — Consolidation & collecte KPI      →  P3 + P8  (phase 2)
Module C — Cadre coaching acheteurs           →  P7       (phase 2)
```

### Module A — Pilote "Excel & data-prep"
- Cartographie de 1-2 fichiers Excel critiques
- Nouvelle version structurée + automatisations simples
- Contrôles de cohérence visibles
- Mini-guide d'usage + session de prise en main
- **Impact attendu** : 60-80% de data-prep automatisée, décisions prises avec plus de sérénité

### Module B — Consolidation & collecte KPI
- Modèle de données global minimal
- Kit reporting standard pour les équipes locales
- Formulaires standardisés + rappels automatiques (écosystème Microsoft)

### Module C — Cadre coaching acheteurs
- Rituels, supports et contenus pour animer la communauté achats
- Amplifier le rôle de "fluidificateur" de Matthieu sans charge supplémentaire

## Méthodologie

- **Framework** : Mom Test (Rob Fitzpatrick) — focus sur les comportements passés, pas les intentions futures
- **Scoring** : 5 dimensions pondérées (douleur ×1, fréquence ×1, impact ×1, urgence ×0.75, WTP ×0.5)
- **Triangulation** : 4 sources de transcription croisées (Fireflies Desktop, Fireflies Browser, Fathom, réconciliation manuelle)
- **Flags Mom Test** : questions orientantes et future-talk identifiés et documentés
